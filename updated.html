What's new

  • Workaround for character display problems (root cause is a Google Chrome bug reported here)
  • Fix to occasional non-removal of deleted items (and possibly other issues relating to feed refresh)
  • New "checkmark" tool for marking feeds. The "spectacles" tool now only opens items
  • Experimental implementation of context menus in all the logical places. Menu items are only in English at the moment
See the documentation for more information.

Known Issues

  • Google Chrome milestone 17 introduced a problem that affects RSS Live Links when a feed URL has been blacklisted by Google. See the Release notes for more information and how to work around the problem.
  • RSS LL Bookmarks and Google sync do not work well together due to Google sync restrictions. Use XMarks sync instead. See the help information for more details.
